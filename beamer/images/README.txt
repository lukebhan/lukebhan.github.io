Lecture highlight figures.

topic-*.tex  : standalone TikZ sources (one per lecture)
topic-*.pdf  : compiled vector figures, used as \lectureimage in the thumbnail
topic-*.png  : 300 dpi transparent rasters, for web use
_figpre.tex  : shared preamble (palette, fonts, TikZ styles)

Drop new per-lecture topic figures here too, then point \lectureimage at one.
