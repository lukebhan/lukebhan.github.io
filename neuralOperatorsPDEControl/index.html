Neural Operators for Accelerated Gain Computation in PDE Control
Luke Bhan, Yuanyuan Shi, and Miroslav Krstic
[arXiv] [code] [tutorial]
  • We replace the kernel computation in PDE backstepping with a learned neural network.
  • Given a Neural Operator gain kernel approximation:$$|\mathcal{K}-\mathcal{K}_\mathbb{N}| \leq \epsilon$$with the following error bound:$$\epsilon^*(B) = \frac{2}{e(1+B)}$$where $|\beta|_\infty \leq B$ for approximating a hyperbolic PDE with gain function coefficient $\beta$ />, then we guarantee the backstepping feedback law satisifies global exponential stability:$$\|u(t)\| \leq Me^{-c^*t/2}\|u(0)\| \quad \forall t \geq 0$$with $M,c^*$ as constants defined in the paper.
Many more details can be found in the tutorial and for the proofs please refer to the paper. If you have any questions, feel free to reach out to the author: lbhan@ucsd.edu.
