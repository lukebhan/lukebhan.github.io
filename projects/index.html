Here are a bunch of personal projects I have either built or collaborated on. They are usually educational exercises (MIT License) and are all available on my github. These projects are listed from most recent start date as a software project never ends :).

Chord, a distributed hash table

July 2020

Chord, available here.

  • Chord is a distributed hash table using the famous protocol outlined here. It allows users to store data(key/value stores) across multiple nodes with O(logn) lookup time.
  • This implementation was written in C++.

Lit, a git implementation

June 2020

Lit, available here.

  • This project is an implementaiton of git using Rust. It has the ability to make simple git commands such as commit as well as the ability to push to github.

Slide, a static expression optimizer

April 2020

Slide, available here. This project was built in collaboration with ayaz hafiz. Check him out!

  • This project allows users to specify a set of rules to simplfiy their mathematical expressions such as x+0 -> x
  • Similar to wolfram alpha; however, ultimately users will select the rules they want to simplify in a general form such as a+0 -> a will simplify all situations where 0 is added to just the variable.
  • This project is completely written in Rust with the UI being written using Golang and Vue.js.

Numlib, an arbitrary precision numerical library

April 2020

Numlib, available here.

  • Numlib is an aribitray precision library for integers and floats implemented in rust.
  • Numlib is a zero-dependency library including its own Fast Fourier Transform, Complex Number implementation and of course, an abitrary precision integer.
